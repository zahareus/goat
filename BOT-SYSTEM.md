# GOAT Bot System

## Що це

Боти — це штучні менеджери, які автоматично обирають гравців на кожен тур. Вони з'являються в standings поруч зі справжніми гравцями. Інші юзери не знають, що це боти.

## Адмін-панель

**Menu → Bot Admin** (видно тільки zahareus@gmail.com)

### Додати бота
1. Введи назву команди (наприклад, "Penalty Kings")
2. Обери стратегію з випадаючого списку
3. Натисни **+ Add Bot**
4. Бот створюється миттєво. `hours_before` генерується рандомно (2-24 години)

### Поставити на паузу
Кнопка **Pause** — бот не подаватиме піки на наступний тур. Його результати залишаються в standings. Кнопка міняється на **Resume** для відновлення.

### Видалити бота
Кнопка **Delete** — бот зникає зі списку. Всі його історичні результати та піки зберігаються в standings (він просто виглядає як звичайний менеджер, без позначки бота).

## 13 стратегій

| ID | Назва | Логіка |
|---|---|---|
| `form` | Form Chaser | Рандом з топ-3 available гравців за формою (avg BPS за останні 6 GW) |
| `goat` | GOAT Hunter | Рандом з топ-3 за кількістю разів GOAT |
| `rank` | Rank Master | Рандом з топ-3 за Bayesian avg rank (менше = краще) |
| `home` | Home Bias | Тільки домашня команда, топ-3 за формою |
| `away` | Away Specialist | Тільки виїзна команда, топ-3 за формою |
| `streak` | Streak Rider | Гравці з висхідним BPS трендом за останні 3 тури |
| `ironman` | Iron Man | Топ-3 за загальною кількістю зіграних хвилин |
| `contrarian` | Contrarian | З топ-5 за формою обирає тих, кого ніхто не обрав |
| `combo` | Balanced | Зважений скор: form×0.4 + goats×0.3 + rank×0.3 |
| `fwd_only` | Striker | Тільки нападники, топ-3 за формою |
| `mid_only` | Midfield Engine | Тільки півзахисники, топ-3 за формою |
| `def_only` | Clean Sheet | Тільки захисники + воротарі, топ-3 за формою |
| `chaos` | Chaos Agent | Повністю випадковий available гравець |

## Як боти обирають

1. Бот дивиться на всі `scheduled` матчі найближчого GW
2. Для кожного матчу збирає гравців обох команд
3. Відсіює недоступних (chance_of_playing < 75% за даними FPL)
4. Відсіює тих, хто жодного разу не грав цього сезону
5. Застосовує стратегію — отримує 3 кандидати
6. Обирає одного рандомно з цих 3

## Коли боти подають піки

Кожен бот має параметр `hours_before` (2-24) — за скільки годин до першого кікоффу туру він подає піки.

**n8n workflow "GOAT Bot Picks"** (`so4OiXG3rd3LqShU`) запускається кожні 30 хвилин і перевіряє:
- Скільки годин до дедлайну найближчого GW
- Які боти мають `hours_before >= hoursToDeadline`
- Які з них ще не подали піки

Боти з'являються поступово протягом дня перед туром — спочатку ті, що за 24h, потім за 18h, 12h і т.д. Це створює ефект реальних гравців, які заходять в різний час.

## Позначка в standings

Коли залогінений zahareus@gmail.com, біля кожного бота в standings (GW та Season) відображається 🤖. Для всіх інших юзерів боти виглядають як звичайні менеджери.

## Технічні деталі

### Файли
- `api/bot-picks.js` — генерація піків (POST, захищений GOAT_NOTIFY_SECRET)
- `api/bot-admin.js` — CRUD для ботів (захищений Supabase auth token)
- `sql/bot-migration.sql` — DDL для колонок

### Колонки в profiles
- `is_bot` (BOOLEAN) — чи це бот
- `bot_active` (BOOLEAN) — чи активний (pause/resume)
- `bot_strategy` (TEXT) — ID стратегії
- `hours_before` (INTEGER) — за скільки годин до дедлайну подає піки

### Ручний запуск (для тестування)
```bash
curl -X POST "https://goatapp.club/api/bot-picks?secret=$GOAT_NOTIFY_SECRET"
```
(Secret зберігається в Vercel env vars та n8n workflow)
